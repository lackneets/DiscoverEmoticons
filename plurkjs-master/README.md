plurkjs
==============================
A javascript command line interface for plunk

HowToRun
==============================

  * Modify lib/key.js for your consumer key/secret
  * call API example:
  
  ` $ node lib/plurk.js '/path/to/API' 'json_params'`
  
  ` $ node lib/plurk.js '/APP/Profile/getPublicProfile' '{"user_id": "clsung"}'`
